# JobHoper

A simple Job Application mobile app that allows users to create an account, search for jobs, and manage their profile.

---

## Features

### Authentication
- User Sign Up (Register new account)
- User Login (Email & Password)
- Password validation
- Forgot Password option
- Logout functionality

### Home Page
- View job listings
- Recently posted jobs
- Job cards showing:
  - Job Title
  - Company Name
  - Location
  - Salary (if available)

### Search
- Search jobs by title
- Search jobs by location
- Filter jobs:
  - Full-time / Part-time
  - Remote / On-site
  - Salary range
- Display search results
- “No results found” message

### Profile
- View personal information
- Edit profile details
- Upload profile picture
- Logout option

## Usage

Follow the steps below to use the Job Application App:

**1. Create an Account**
- Open the app.
- Click on **Sign Up**.
- Enter your full name, email, and password.
- Verify your email (if required).
- Successfully create your account.

**2. Login**
- Enter your registered email and password.
- Click **Login** to access the app dashboard.

**3. Browse Jobs (Home Page)**
- View available job listings.
- Check job details such as title, company, location, and salary.
- Select a job to see more information.

**4. Search for Jobs**
- Use the search bar to find jobs by title or location.
- View matching results.

**5. Manage Your Profile**
- Go to the **Profile** section.
- Update personal information.

**6. Logout**
- Click the **Logout** button when finished using the app.
